Diccionario de términos

🌐 Máquina virtual
Software que simula un ordenador físico dentro de otro sistema, permitiendo instalar y usar sistemas operativos de forma aislada.

🔌 Dirección IP
Número que identifica de forma única a un dispositivo dentro de una red, permitiendo su comunicación con otros equipos.

🔐 Permisos en Linux
Reglas que determinan qué puede hacer cada usuario con un archivo o carpeta: leer, escribir o ejecutar.

🧩 Samba
Software que permite compartir archivos e impresoras entre sistemas operativos diferentes, como Linux y Windows.

📂 Recurso compartido
Carpeta, archivo o dispositivo que se pone a disposición de otros usuarios dentro de una red.
