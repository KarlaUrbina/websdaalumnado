Sda Propuesta para el alumnado

En esta situación de aprendizaje vamos a aprender cómo compartir archivos entre un ordenador con Linux y otro con Windows utilizando Samba. Esto es muy útil cuando queremos que varios equipos, aunque tengan sistemas operativos diferentes, puedan acceder a los mismos archivos dentro de una red.

Primero veremos qué es Samba y para qué sirve. Después, aprenderemos a instalarlo y configurarlo en Linux para compartir carpetas. Por último, veremos cómo acceder a esas carpetas desde un ordenador con Windows de forma sencilla.

El objetivo es que seas capaz de compartir y utilizar archivos en red entre distintos sistemas, algo muy común en entornos reales de trabajo.
